TROMPA CE client
================

The TROMPA CE client is a Python library that can be used to read data from and write data to the
TROMPA Contributor Environment using its GraphQL interface.

Installation
------------

To install the package, use pip::

    pip install trompace-client


For information about the TROMPA schema, see :ref:`schema:TROMPA Schema`
